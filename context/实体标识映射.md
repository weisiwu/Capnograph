## 手写备注：Packflow 接入

- Packflow 后台配置不是 `context-seed` 生成的源码实体；定位入口见 `context/entity-id-mapping.md` 的 `Packflow backend configs` 行，以及 `context/docs/platform-resources/PACKFLOW.md`。
- Packflow Android APK 60 秒目标、失败记录和优化清单见 `context/docs/packflow-android-apk-goal.md`。
- `capno_packflow_bot_command` 是飞书原始文本口令入口，`capno_packflow_agent` 是打包入口，`capno_packflow_query` 是历史/详情/配置查询入口；`CapnoGraph OMP Bot` webhook 配置只通过环境变量注入，不写入仓库。
- Android debug APK 当前约 `105690613` bytes；飞书交付走 Packflow 公开 token 下载链接，不上传 APK 文件本体。外网可访问链接通过 `CAPNOGRAPH_PACKFLOW_PUBLIC_BASE_URL` / `PACKFLOW_PUBLIC_BASE_URL` 配置，或从本机 `~/.cloudflared/packflow-baoganai.yml` 自动发现 `https://packflow.baoganai.com`；APK 文件名追加版本和时间。
