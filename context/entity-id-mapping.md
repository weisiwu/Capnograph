# CapnoEasy 实体 ID 定位表

这个定位文件用于把用户可见名称、代码实体、协议 ID、存储 ID 和资源 ID 映射到对应源文件。回答项目相关问题或修改代码前，先从这里定位上下文。

当前可用的补充上下文：

- 项目记忆主文档：`.cursor/rules/project-memory.mdc`
- 构建与基础能力：`context/docs/build-platform.md`
- 平台、资源、清单与测试入口实体文档目录：`context/docs/platform-resources/`（索引：`context/docs/platform-resources.md`）
- 应用入口、页面、UI 与状态实体索引：`context/docs/app-ui-workflows.md`（实体文档目录：`context/docs/app-ui-workflows/`）
- 核心数据、协议与业务服务索引：`context/docs/core-data-protocol.md`（实体文档目录：`context/docs/core-data-protocol/`）
- 数据库版本说明：`app/data_version_list.txt`
- Room schema 快照：`app/schemas/com.wldmedical.capnoeasy.kits.AppDatabase/1.json`
- 其他实体级补充文档可继续补充到 `context/docs/`。没有对应补充文档时，请读取本表映射的源文件和项目记忆主文档。

## 使用方式

1. 将用户提到的显式名称、中文显示名、类名、枚举值、ID、UUID、表名、偏好设置 key 或资源名与本文件匹配。
2. 读取匹配行里的 `源文件`。
3. 仅在匹配行写有 `补充上下文` 时读取对应文档。文档和代码不一致时，以当前代码为准。
4. 如果改动新增、删除、重命名或移动了本文件列出的实体或 ID，需要同步更新本文件。
